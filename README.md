# STAT3008 Group Project

Group project on Gaussian Processes (GPs) for STAT3008 (Selected Topics in Statistical Learning) at UQ in 2025, Sem 2.

Authors:

- Ethan Jones
- Angus Scroggie
- Jay Parfitt

## Report

The report includes a mathematical background to GPs, problem formulation, description of code, and example analysis of artificial and real time series data using GPs.

## Code

Jupyter notebooks contain code used to perform data analysis and model fitting for the report.
This includes: artificial time series data generation, weather data EDA, and GP model fitting and inference.

## Data

We used hourly historical weather data for Brisbane from OpenWeather. This is available for purchase [here](https://home.openweathermap.org/history_bulks/new).
